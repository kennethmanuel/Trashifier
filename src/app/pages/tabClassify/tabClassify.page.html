 What is Trashifier?
Trashifier is an app created to help people recognize their garbage type and be more informed about sorting and recycling their trash.
How do I use Trashifier?

How does Trashifier work?
Trashifier work by using a machine learning model specifically designed to classify a trash, hence the name Trashifier.

The app simply sends the image to a server in which the classifier is deployed. The server then tries to predict and sent back the result. Note that Trashifier does not collect any of the image sent.
